# 💰 EduInvest AI - Assessor de Investimentos Virtual

O **EduInvest AI** é um chatbot inteligente desenvolvido para democratizar o acesso ao conhecimento sobre educação financeira e investimentos. O projeto utiliza a API do Google Gemini para fornecer respostas rápidas, objetivas e seguras para investidores iniciantes.

---

## 🚀 Funcionalidades
- **Interação em Tempo Real:** Interface de chat fluida e responsiva.
- **Eduardo (Persona AI):** Um assessor virtual personalizado com foco em clareza e didática.
- **Sugestões Inteligentes:** O chatbot sugere próximos passos para aprofundar o conhecimento do usuário.
- **Design Moderno:** Layout focado em UX (User Experience) com elementos visuais profissionais.

---

## 🛠️ Tecnologias Utilizadas
- **Backend:** Python 3.10+ com [Flask](https://flask.palletsprojects.com/)
- **Frontend:** HTML5, CSS3 (Glassmorphism design) e JavaScript Assíncrono.
- **Inteligência Artificial:** [Google Gemini API](https://ai.google.dev/) (Modelo 3.0 Flash)
- **Segurança:** Variáveis de ambiente com `python-dotenv`.

---

## 🎯 Casos de Uso
Este projeto foi idealizado para ser escalável e pode ser implementado em:
- **Casas de Research:** Para automatizar a explicação de conceitos macroeconômicos e relatórios.
- **Assessorias de Investimento:** Como ferramenta de *onboarding* educativo para novos clientes.
- **Fintechs:** Para suporte de primeiro nível sobre produtos financeiros.
